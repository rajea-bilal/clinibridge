Download the official logos and place them in this directory:

1. **tanstack.svg**
   - Download from: https://tanstack.com or their GitHub repo
   - Format: SVG (preferred) or PNG
   - Colors: Should work on dark backgrounds

2. **better-auth.svg**
   - Download from: https://better-auth.com or their GitHub repo
   - Format: SVG (preferred) or PNG
   - Colors: Should work on dark backgrounds

3. **polar.svg**
   - Download from: https://polar.sh/brand or their press kit
   - Format: SVG (preferred) or PNG
   - Note: Get the white/light version for dark backgrounds

4. **convex.svg**
   - Download from: https://convex.dev or https://docs.convex.dev
   - Format: SVG (preferred) or PNG
   - Note: Get the version that works on dark backgrounds

5. **resend.svg**
   - Download from: https://resend.com/brand or their press kit

   - Format: SVG (preferred) or PNG

7. **ruler.svg**
   - Download from: https://github.com/intellectronica/ruler
   - Format: SVG (preferred) or PNG
   - Note: Get the light version for dark backgrounds

8. **ultracite.svg**
   - Download from: https://www.ultracite.ai or their brand assets
   - Format: SVG (preferred) or PNG
   - Note: Get the light version for dark backgrounds

## Tips

- Prefer SVG format for scalability
- If using PNG, aim for at least 512x512px
- Ensure logos work on dark backgrounds (white/light versions)
- Keep file names lowercase with hyphens
- If a logo doesn't have an SVG, PNG works fine
