# Yugen 幽玄

Monorepo Boilerplate, built with TanStack Start, Convex, BetterAuth, Polar.sh payments, Resend emails, and Cloudflare Workers deployment (w/ Alchemy).

## Features

- ✅ **Authentication**: BetterAuth with email/password
- ✅ **Waitlist**: Optional waitlist functionality (configurable)
- ✅ **Database**: Convex (real-time serverless database)
- ✅ **Payments**: Polar.sh integration with subscriptions
- ✅ **Emails**: Resend integration with welcome emails
- ✅ **File Storage**: Cloudflare R2 (97% cheaper than Convex storage)
- ✅ **Frontend**: TanStack Start (React Router)
- ✅ **Deployment**: Cloudflare Workers via Alchemy

## In-Codebase Documentation

```bash
bun install
cd apps/fumadocs
bun run dev
```

**Docs available at:** `http://localhost:4000` ✅

For the complete app setup with authentication, payments, etc., follow the 'Setup' guides

The docs are also in both the Accelerator AND published live to: https://yugen-fumadocs.vercel.app
